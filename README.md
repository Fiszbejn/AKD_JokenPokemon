# AKD JokenPokemon 🕹️⚡

Aplicativo Android desenvolvido em **Kotlin** utilizando **Jetpack
Compose**, que implementa um jogo simples inspirado em **Jokenpô**, com
temática de **Pokémon**.

O usuário escolhe um Pokémon através de botões na interface e a
aplicação executa uma lógica que define se o jogador **ganhou, perdeu ou
empatou** contra a escolha do adversário (CPU).

------------------------------------------------------------------------

## 🎯 Objetivo do Projeto

-   Praticar desenvolvimento de aplicativos Android utilizando
    **Kotlin**
-   Aplicar **Jetpack Compose** para criação de interfaces declarativas
-   Implementar lógica de decisão (vitória/derrota/empate)
-   Evoluir organização e boas práticas em um projeto Android

------------------------------------------------------------------------

## 🧩 Funcionalidades

-   Interface construída com **Jetpack Compose**
-   Seleção de Pokémon via botões
-   Escolha aleatória do adversário (CPU)
-   Cálculo do resultado da rodada:
    -   Vitória
    -   Derrota
    -   Empate
-   Atualização dinâmica do resultado na interface

------------------------------------------------------------------------

## ▶️ Como executar o projeto

### 1. Clonar o repositório

``` bash
git clone https://github.com/Fiszbejn/AKD_JokenPokemon.git
```

### 2. Abrir no Android Studio

Abra o projeto no **Android Studio**.

### 3. Sincronizar dependências

Aguarde o **Gradle Sync** finalizar automaticamente.

### 4. Executar o aplicativo

Execute o app em: - um **emulador Android** - ou um **dispositivo
físico**
